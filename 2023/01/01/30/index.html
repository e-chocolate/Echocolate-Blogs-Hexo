Git同步两个远程仓库

　　有时候为了保证项目代码的安全，会将本地项目都在不同的Git平台都有备份，具体操作如下：

  1. 查看当前本地项目已经关联的远程仓库
  1. 为当前项目新关联另一个远程仓库，要注意新关联的name不能与已经有的相同
  1. 再次查看本地项目的已经关联的数据库
  1. 针对不同的远程库执行推送
    git push [-u] gitlab_origin main
    git push [-u] origin main
