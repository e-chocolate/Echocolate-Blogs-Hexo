Frp进阶——为内网服务启用域名访问

Frp 是一款高效的内网穿透工具。利用该工具可以暴露内网服务，非常实用。

官方文档介绍了 Frp 的一些使用场景，其中最常见的就是暴露内网 ssh 端口。

Frp 同时还支持为内网的 Web 服务提供域名访问的方式，既支持http协议，也支持https协议。不过，这样每次访问都要通过域名 + 端口 来进行访问。因此可以利用 nginx 来反向代理 Frp 的服务，从而实现域名访问的效果，需要在 nginx 的 server 段添加下面的配置项:
